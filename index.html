Core features

Standard command replies

The bot can answer commands like !tg, !discord, !donate, or any other custom command created for the channel.

Command spam mode

Using a format like !tg3, the bot repeats the answer of !tg three times in a row.

Phrase spam mode

The bot can also spam plain text with a format like !spam hello 5, sending the word hello five times.

New commands from chat

Commands can be added directly from Twitch chat with !cmd add !test text, making updates easier for the streamer.

Reply tag

If the saved response ends with #reply#, the bot sends it as a reply to the original chat message.

Mention placeholder

The placeholder %user% becomes a real mention like @Fly when the command is used.
